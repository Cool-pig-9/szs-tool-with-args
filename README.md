# SZS-Tool
Compress and decompress SZS files with the Yaz0 compression

## Compatibility
**FOR USE WITH NINTENDO 3DS GAMES ONLY**. The `.szs` files from a Wii game **will not work**. (and intended for use on super mario 3d land)
